Manage your account

Create a sign-in you can use for the community forum. Accounts and sessions are stored directly in this repository through the GitHub API.

Connect to GitHub

GitHub Pages is static, so writing data requires a personal access token with repo scope. The token is only stored in your browser and used to commit account and forum changes.

Create a new account

Use a unique username and a password with at least eight characters. Your email helps identify the account in the saved repository data.

Sign in

Already created an account? Sign in with your email or username and password. Your credentials are validated against the repository data store.

Not signed in.

Heads up: your browser blocked local storage, so your sign-in will reset after you close this tab.

Need to start posting? Head back to the community forum.
